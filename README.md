# Фронтенд для дипломного проекта Movies Explorer

### Используемые технологии:

- HTML5, CSS3
- методология БЭМ
- Адаптивная вёрстка
- ReactJS, JSX

## Запуск проекта
`npm i` - устанавливает зависимости

`npm start` — запускает проект локально
